# WordPress Folder Media Gallery Templates

**Zero-plugin, high-performance way to browse thousands of images and videos directly from server folders** — no Media Library bloat.

Perfect for photographers, videographers, backup archives, private media servers, and large collections.

---

## ✨ Features

- **No WordPress Media Library import** — reads files directly from any folder (including symlinks)
- Supports **Images** + **Videos** + **Mixed folders**
- Clean responsive grid with lightbox
- Proper **pagination** (handles 1,000–10,000+ files easily)
- Subfolder navigation
- Video thumbnails generated with **ffmpeg**
- Uses **Custom Fields** — super easy to manage multiple galleries
- Symlink friendly (best performance & security)

---

## 📁 Included Templates

| Template Name                          | Purpose                          | Custom Field     |
|---------------------------------------|----------------------------------|------------------|
| `video-grid-catalog.php`              | Videos only                      | `video_folder`   |
| `image-grid-custom-field.php`         | Images only (lightbox)           | `image_folder`   |
| `media-grid-mixed.php`                | **Images + Videos** together     | `media_folder`   |
